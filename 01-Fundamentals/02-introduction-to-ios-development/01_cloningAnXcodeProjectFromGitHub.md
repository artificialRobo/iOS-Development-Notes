# Notes: Cloning an Xcode Project from GitHub

## What is GitHub?

  * Upload and share code
  * Collaborate on projects
  * Manage code changes efficiently

---

## How to Clone a Project from GitHub

### Step 1: Open the Course Resources

* Go to the Dicee module resources.
* Click **Dicee Starting Project**.
* This opens the project repository on GitHub.


### Step 2: Copy the Repository Link

* Click the **Clone or download** button.
* Copy the repository URL.
* Important:

  * The repository URL is different from the webpage URL.
  * The copied link is specifically for cloning the project.


### Step 3: Open Xcode

* Close any currently open projects.
* Xcode automatically saves work frequently, so losing progress is rare.

You can access cloning from:

1. **Source Control → Clone**
2. Xcode welcome screen → **Clone an existing project**

Both options do the same thing.


### Step 4: Clone the Repository

* Paste the copied GitHub link into Xcode.
* Click **Clone**.
* Download time depends on internet speed.
* Choose a save location for the project.
* You may rename the project if desired.

## Flappy Bird Challenge

Students are encouraged to:

1. Clone the Flappy Bird Swift project
2. Run it in the iOS simulator
3. Play and explore the code

### Recommendation:

* Run internet-downloaded projects only in the **simulator**.
* Running on a real device requires:

  * Signing setup
  * Unique bundle identifier
  * Apple developer configuration
