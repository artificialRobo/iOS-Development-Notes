## BMI Calculator Module

### Overview

In this module, you will build a **BMI (Body Mass Index) Calculator App** and learn several important iOS development concepts.

### Learning Objectives

By the end of the module, you will be able to:

* Build a functional and visually appealing BMI Calculator app.
* Calculate and display a user's BMI.
* Implement and use a **UI Slider**.
* Create **multi-screen apps** using **segues**.

### Swift Concepts Covered

#### Classes

* Learn how to create and use **classes**.
* Understand the differences between **classes** and **structs**.

#### Optionals

Gain a deeper understanding of optionals, including:

* **Optional Binding**
* **Optional Chaining**
* **Nil Coalescing Operator (`??`)**

### User Interface Topics

#### Programmatic UI

* Create user interfaces **without using Storyboards**.
* Build UI elements directly in code.

#### Navigation

* Learn how to move between screens using **segues**.

### Additional Features

#### Color Literals

* Use **Color Literals** to select colors directly from a color palette in code.
* Similar to **Image Literals**, but for colors.

### Final Project Features

The BMI Calculator app will include:

* BMI calculation functionality.
* Interactive slider controls.
* Multiple screens.
* Custom UI design.
