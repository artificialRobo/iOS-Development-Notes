# Notes: How to Use Canva to Create Mockups

## Canva as an Alternative

* If you don't have **Sketch**, you can use **Canva** to create app mockups.
* Canva is useful for:

  * Posters
  * Social media graphics
  * Beautiful layouts
  * App mockups
* It offers many **editable templates** and a simple **drag-and-drop** interface.

## Setting Up the Canvas

1. Go to **Canva.com**.
2. Choose **Custom Dimensions**.
3. Set the canvas size to **750 × 1344 px** (iPhone screen size).
4. Click **Design** to create a blank canvas.

## Creating Wireframes

* Search for **"wireframe"** in the Elements/Search section.
* Drag and drop UI components onto the canvas.
* Most components are **free**, while some are paid.

## Designing the App

* Add interface elements such as:

  * Navigation back button
  * Lists
  * Squares, circles, and other shapes
* Resize and position elements as needed.
* Customize colors (e.g., dark gray navigation bar).
* Add a title like **"My App"**.
* Change fonts to improve appearance.

## Creating Multiple Screens

* Click **Add New Page** to create another app screen.
* Each new page keeps the same dimensions.
* Continue designing with drag-and-drop components.

## Customization Features

* Access:

  * Color palettes
  * Backgrounds (e.g., honeycomb patterns)
  * Text styles
  * Icons (e.g., profile icons)

## Limitations

* Canva is mainly designed for **print design**, not UI design.
* It lacks some UI elements like:

  * Status bars
  * Navigation bars

## Workaround for Missing UI Elements

* Search Canva for needed images.
* If unavailable:

  * Search Google for the required UI element.
  * Download it.
  * Upload it into Canva.
  * Drag and drop it into your design.

## Sharing & Exporting

* Share Canva projects with others for collaboration.
* Export designs as:

  * Images
  * PDFs

## Key Takeaway

Canva is a **quick, easy, and beginner-friendly** alternative to Sketch for creating simple app mockups using drag-and-drop tools, templates, icons, and customizable design elements.
