# Notes: Color Palette Tools for App & Web Design

## Key Idea

* Understanding color theory is useful, but app developers and entrepreneurs often need faster ways to create appealing color palettes.
* Several tools can help generate or discover professional-looking color schemes without manually experimenting with colors.

---

## 1. Color Hunt

**Website:** Color Hunt

**Purpose:**

* Provides curated color palettes created by professional designers.
* Useful for finding palettes that match a project's mood or style.

**Features:**

* Browse palettes by:

  * Hot
  * Popular
  * Random
* Large collection of ready-to-use color combinations.

**Best for:**

* Designers looking for inspiration and professionally curated palettes.

---

## 2. Flat UI Colors

**Website:** Flat UI Colors

**Purpose:**

* Offers a predefined 15-color palette optimized for iOS app design.

**Key Points:**

* Many iOS apps use these exact shades and tones.
* Colors work well together and fit the iOS aesthetic.
* Example: The Seven Minute Workout app uses a similar palette.

**Best for:**

* iOS developers who want a quick, reliable color scheme.

---

## 3. Material Palette

**Website:** Material Palette

**Purpose:**

* Designed for Android app development.

**Features:**

* Select two colors.
* Automatically generates:

  * An 8-color palette
  * Accent color/tone

**Benefits:**

* Makes creating Android-compatible color schemes fast and easy.

**Best for:**

* Android designers following Material Design principles.

---

## 4. ColorZilla

**Tool:** Free Chrome & Firefox extension

**Purpose:**

* Acts as a color picker (eyedropper) within the browser.

**Features:**

* Detects colors from any website.
* Reveals the exact hex code of a color.
* Allows saving colors for future projects.

**Best for:**

* Finding and reusing colors from websites with appealing designs.

---

## Key Takeaway

By combining:

1. Color theory,
2. Knowledge of color moods and emotions,
3. Color palette tools (Color Hunt, Flat UI Colors, Material Palette, and ColorZilla),

you can quickly create attractive color palettes that:

* Improve app appearance,
* Create a consistent design,
* Evoke specific emotions or ideas in users.
