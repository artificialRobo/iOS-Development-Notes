# Notes: Title Placement & Dropdown Differences (iOS vs Android)

## 1. Title Placement

**Android**

* Screen titles are **left-aligned**.
* Text extends toward the right.

**iOS**

* Screen titles are **center-aligned**.
* This is the standard design convention on iOS.

**Why it Matters**

* Small UI details help users feel that an app is designed specifically for their platform.
* Ignoring these conventions can make an app feel poorly designed or simply copied from another platform.

---

## 2. Dropdown Menus

**Android**

* Uses a **dropdown menu**.
* Users tap a selector to display a list of options.

**iOS**

* Does **not** use Android-style dropdown menus.
* Instead, it commonly uses a **sliding wheel (picker)** to select options.
* The picker has a different appearance, animation, and interaction style.

---

## Key Takeaways

* **Android titles:** Left-aligned.
* **iOS titles:** Center-aligned.
* **Android:** Uses dropdown menus for option selection.
* **iOS:** Uses a sliding wheel (picker) instead of dropdown menus.
* Following each platform's UI conventions creates a more natural and user-friendly experience.
