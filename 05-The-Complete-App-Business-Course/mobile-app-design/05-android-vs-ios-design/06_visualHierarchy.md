# Notes: Visual Hierarchy (iOS vs Android)

## What is Visual Hierarchy?

* Visual hierarchy is the arrangement of elements based on their importance.
* It helps guide the user's attention from the **most important** information to the **least important**.

---

## Android

* Visual hierarchy is created using **text size**.
* Common pattern:

  * **Title:** Largest text
  * **Subtitle:** Medium-sized text
  * **Body text:** Smallest text

---

## iOS

* Visual hierarchy is created using **font weight (thickness)** instead of size.
* Common pattern:

  * **Title:** Bold
  * **Subtitle:** Regular
  * **Body text:** Light/Thin

---

## Key Takeaways

* **Android:** Uses **different text sizes** to indicate importance.
* **iOS:** Uses **different font weights** (bold, regular, light) to indicate importance.
* A clear visual hierarchy improves readability and naturally directs the user's attention through the interface.
