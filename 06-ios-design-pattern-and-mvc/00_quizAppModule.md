## Module Introduction: Structs & Model-View-Controller (MVC)

### Overview

This module focuses on **Swift Structures (Structs)** and the **Model-View-Controller (MVC) Design Pattern**, two important concepts in iOS app development.

### What You'll Build

A **Quiz App** that:

* Displays quiz questions
* Lets users select answers
* Tracks quiz progress
* Shows the user's score

### Key Learning Objectives

#### 1. Model-View-Controller (MVC) Design Pattern

* Learn a common design pattern used in iOS development.
* Understand how to organize app code using:

  * **Model** – manages data and business logic
  * **View** – displays information to the user
  * **Controller** – handles interactions between the Model and View

#### 2. Swift Structures (Structs)

* Learn what structs are and how to use them in Swift.
* Understand how structs fit into design patterns like MVC.

#### 3. Quiz App Functionality

* Display quiz questions to users.
* Check whether selected answers are correct.
* Update and display progress and scores.

#### 4. Functions with Inputs and Outputs

* Create functions that:

  * Accept parameters (inputs)
  * Return values (outputs)

#### 5. Immutability in Swift

* Learn the concept of **immutability**.
* Understand when data can and cannot be changed.

### Outcome

By the end of the module, you will be able to:

* Build a more complex iOS app.
* Apply the MVC design pattern.
* Use Swift structs effectively.
* Write functions with parameters and return values.
* Understand and work with immutable data in Swift.
